## 2. 領域別スコアリング

### 2.1 睡眠（ESS: Epworth Sleepiness Scale）

**算出方法**: 8項目の合計点（各0〜3点）

```
スコア = Σ items[0..7]  (範囲: 0〜24点)
```

**判定ツリー**:

```
ESS合計点
├── 0〜10点 → LOW（良好）
│   ├── referralNeeded: false
│   ├── 解釈: 日中の眠気は正常範囲内
│   └── 推奨:
│       ├── 睡眠時間(7〜8時間)の維持
│       ├── 就寝・起床時刻の固定
│       └── 運転前の十分な睡眠
│
├── 11〜15点 → MODERATE（要改善）
│   ├── referralNeeded: true ⚠️
│   ├── 受診勧奨: SASスクリーニング検査
│   ├── 解釈: 過度の日中眠気 → SAS疑い
│   └── 推奨:
│       ├── 睡眠外来の受診検討
│       ├── カフェイン・アルコール制限
│       ├── 寝室環境の整備
│       └── 15〜20分の仮眠活用
│
└── 16〜24点 → HIGH（要注意）
    ├── referralNeeded: true 🚨
    ├── 受診勧奨: PSG精密検査 → CPAP検討
    ├── 解釈: 重度の日中眠気 → 居眠り運転リスク高
    └── 推奨:
        ├── 早急に睡眠外来を受診
        ├── PSG検査の実施
        ├── 長距離運転の制限検討
        └── 産業医と就業制限を相談
```

### 2.2 ストレス（K6: Kessler Psychological Distress Scale）

**算出方法**: 6項目の合計点（各0〜4点）

```
スコア = Σ items[0..5]  (範囲: 0〜24点)
```

**判定ツリー**:

```
K6合計点
├── 0〜4点 → LOW（良好）
│   ├── referralNeeded: false
│   ├── 解釈: 心理的苦痛は正常範囲内
│   └── 推奨:
│       ├── ストレス対処法(趣味・運動・休息)の継続
│       ├── 相談体制の確認
│       └── 定期的セルフチェック
│
├── 5〜12点 → MODERATE（要改善）
│   ├── referralNeeded: false
│   │   ※ ただし受診勧奨詳細に産業医フォロー推奨を記載
│   ├── 解釈: 心理的苦痛あり → セルフケア+専門家相談推奨
│   └── 推奨:
│       ├── ストレス原因の把握と対処
│       ├── 十分な睡眠と休息
│       ├── 周囲への相談(産業医・保健師)
│       └── リラクゼーション法の導入
│
└── 13〜24点 → HIGH（要注意）
    ├── referralNeeded: true 🚨
    ├── 受診勧奨: 心療内科/精神科 → 就業制限検討
    ├── 解釈: 重度の心理的苦痛 → うつ病・不安障害の可能性
    └── 推奨:
        ├── 心療内科/精神科を受診
        ├── 産業医に報告 → 就業配慮相談
        ├── 信頼できる人に相談
        └── 無理せず十分な休養
```

### 2.3 疲労（厚生労働省 疲労蓄積度自己診断チェックリスト）

**算出方法**: 13項目の合計点（各0〜3点）

```
スコア = Σ items[0..12]  (範囲: 0〜39点)
```

**判定ツリー**:

```
疲労蓄積度合計点
├── 0〜4点 → LOW（良好）
│   ├── referralNeeded: false
│   ├── 解釈: 疲労の蓄積度は低い
│   └── 推奨:
│       ├── 現在の生活リズム・休息パターンの維持
│       ├── 乗務前後の適切な休息
│       └── 定期的なセルフチェック
│
├── 5〜14点 → MODERATE（要改善）
│   ├── referralNeeded: false
│   │   ※ 受診勧奨詳細に産業医フォロー推奨を記載
│   ├── 解釈: 疲労がやや蓄積 → 注意力低下リスク
│   └── 推奨:
│       ├── 睡眠時間(7〜8時間)の確保
│       ├── 連続運転の回避 → こまめな休憩
│       ├── 休日の十分な休息
│       └── 改善しない場合は産業医・保健師に相談
│
└── 15〜39点 → HIGH（要注意）
    ├── referralNeeded: true 🚨
    ├── 受診勧奨: 産業医面談(過重労働面談) + 背景疾患の精査
    ├── 解釈: 高度の疲労蓄積 → 安全運転に支障の恐れ
    └── 推奨:
        ├── 産業医面談の早急な実施
        ├── 勤務時間・運転時間の見直し
        ├── 背景疾患の検査(睡眠障害・うつ病等)
        ├── 十分な休養の確保
        └── 一時的な業務軽減の検討
```

### 2.4 食事・栄養（特定健診質問票準拠）

**算出方法**: 8項目の合計点（各0〜4点）

```
スコア = Σ items[0..7]  (範囲: 0〜32点)
※ 食事は高得点=良好（逆転スコアリング項目あり）
```

**判定ツリー**:

```
食事スコア合計点
├── 20〜32点 → LOW（良好）
│   ├── referralNeeded: false
│   ├── 解釈: 食生活はおおむね良好
│   └── 推奨:
│       ├── 現在の良好な食習慣の維持
│       ├── 季節の野菜・果物の積極摂取
│       └── 水分補給の継続
│
├── 12〜19点 → MODERATE（要改善）
│   ├── referralNeeded: false
│   │   ※ 特定保健指導の対象可能性を記載
│   ├── 解釈: 食生活に改善の余地あり
│   └── 推奨: ← ★ 項目別に動的生成
│       ├── [朝食 ≤1点] 朝食を毎日とる習慣
│       ├── [野菜 ≤1点] 野菜摂取量の増加(1日350g目標)
│       ├── [水分 ≤1点] 水分摂取量の増加
│       ├── [夜食 ≤1点] 夜食・間食の制限
│       ├── [食事時間 ≤1点] 食事時間の規則化
│       ├── [塩分 ≤1点] 減塩食品の活用
│       ├── [飲酒 ≤1点] 適正飲酒 + 休肝日
│       └── [カフェイン ≤1点] 1日400mg以内の制限
│
└── 0〜11点 → HIGH（要注意）
    ├── referralNeeded: true 🚨
    ├── 受診勧奨: 管理栄養士による個別指導 / 特定保健指導(積極的支援)
    ├── 解釈: 食生活に大きな課題 → 生活習慣病リスク高
    └── 推奨: ← ★ 項目別に動的生成（上記と同じロジック、文面強化）
```

**項目別推奨生成ロジック**:

```
for each item in answers.items:
  if item[i] <= 1:
    recommendations.push(対応する改善アドバイス)

if recommendations.length === 0:
  recommendations.push("全体的な食事バランスの改善")
```

| インデックス | 評価項目 | 低スコア時の推奨 |
|------------|---------|-----------------|
| 0 | 朝食頻度 | 朝食の毎日摂取（おにぎり1個からでも） |
| 1 | 野菜摂取 | 1日350g（小鉢5皿程度）を目標 |
| 2 | 水分摂取 | 運転中のこまめな水分補給 |
| 3 | 夜食・間食 | 夕食後の飲食を控える |
| 4 | 食事時間の規則性 | シフト勤務でも食事時間の目安設定 |
| 5 | 塩分摂取 | 高血圧予防のための減塩 |
| 6 | 飲酒 | 適正飲酒量（ビール中瓶1本）+ 週2日休肝日 |
| 7 | カフェイン | 1日400mg以内（コーヒー約4杯） |

### 2.5 運動・身体活動（IPAQ-SF）

**算出方法**: MET-分/週の算出

```
激しい運動MET  = 8.0 × vigorousMinutes × vigorousDays
中程度運動MET  = 4.0 × moderateMinutes × moderateDays
歩行MET       = 3.3 × walkingMinutes  × walkingDays

合計MET = 激しい運動MET + 中程度運動MET + 歩行MET
```

**座位時間警告**: `sittingHours >= 8` で追加推奨を付与

**判定ツリー**:

```
MET-分/週
├── > 3,000 → LOW（良好）
│   ├── referralNeeded: false
│   ├── 解釈: 身体活動量は十分
│   ├── 推奨: 現在の運動習慣の維持
│   └── [座位≥8h] 30分〜1時間ごとにストレッチ
│
├── 600〜3,000 → MODERATE（要改善）
│   ├── referralNeeded: false
│   │   ※ 保健師による運動指導を推奨
│   ├── 解釈: 身体活動量は中程度 → さらなる増加推奨
│   ├── 推奨:
│   │   ├── ウォーキング30分/日を目標
│   │   ├── 休憩時にストレッチ・体操
│   │   └── 日常での活動量増加（階段利用等）
│   └── [座位≥8h] 1時間ごとに立ち上がって体を動かす
│
└── < 600 → HIGH（要注意）
    ├── referralNeeded: true 🚨
    ├── 受診勧奨: 運動指導・生活習慣改善 / 特定保健指導
    ├── 解釈: 身体活動量不足 → 生活習慣病リスク増大
    ├── 推奨:
    │   ├── 1日15分のウォーキングから開始
    │   ├── 休憩時5分間のストレッチ
    │   ├── 週2〜3日の意識的な運動
    │   └── 保健師に相談して運動プラン作成
    └── [座位≥8h] エコノミークラス症候群リスク → 2時間ごとに休憩
```

## 3. 総合リスク判定

```
        ┌─────────────────────────────────────┐
        │    5領域の DomainResult を集約        │
        └───────────────┬─────────────────────┘
                        │
                ┌───────▼───────┐
                │ highCount =   │
                │ HIGH の領域数  │
                └───────┬───────┘
                        │
                  ┌─────▼─────┐
                  │highCount≥1│
                  └─────┬─────┘
                   Yes  │  No
              ┌─────────┤
              ▼         │
     ┌────────────┐     │
     │ 総合: HIGH │     ▼
     │ 要注意     │  ┌──────────────┐
     └────────────┘  │moderateCount │
                     │= MODERATE数  │
                     └──────┬───────┘
                            │
                    ┌───────▼───────┐
                    │moderateCount≥2│
                    └───────┬───────┘
                     Yes    │  No
                ┌───────────┤
                ▼           ▼
       ┌─────────────┐  ┌──────────┐
       │ 総合: MODERATE│ │ 総合: LOW│
       │ 要改善       │ │ 良好     │
       └─────────────┘  └──────────┘
```

**判定ルール**:

| 条件 | 総合リスク | サマリー文 |
|------|-----------|-----------|
| HIGH が **1領域以上** | HIGH | 「{領域名}の領域で要注意…医療機関の受診を強く推奨」 |
| HIGH なし & MODERATE が **2領域以上** | MODERATE | 「複数の領域で改善が望まれる…産業医・保健師との定期面談を推奨」 |
| 上記以外 | LOW | 「全体的に良好…現在の生活習慣を維持」 |

**受診勧奨フラグ**:

```typescript
referralRecommended = domains.some(d => d.referralNeeded)
```

いずれかの領域で `referralNeeded: true` → 結果画面に受診勧奨バナーを表示

## 4. アクションプラン生成ロジック

### 4.1 生成フロー

```
AssessmentResult
├── 領域別アクション生成（×5ドメイン）
│   ├── sleepActions(DomainResult)
│   ├── stressActions(DomainResult)
│   ├── fatigueActions(DomainResult)
│   ├── dietActions(DomainResult)
│   └── exerciseActions(DomainResult)
│
├── クロスドメインアクション生成
│   └── crossDomainActions(AssessmentResult)
│
└── 優先度ソート（urgent → recommended → optional）
```

### 4.2 アクション属性

| 属性 | 型 | 説明 |
|------|-----|------|
| `priority` | `urgent / recommended / optional` | 優先度 |
| `category` | `medical_referral / screening / consultation / lifestyle_guidance / follow_up / self_care` | アクション種別 |
| `actor` | `occupational_physician / specialist / public_health_nurse / nutritionist / exercise_instructor / driver` | 担当者 |
| `timeline` | `string` | 実施期限 |
| `checklist` | `string[]` | 具体的な実施項目 |


### 4.4 クロスドメイン判定ツリー

```
全5領域の判定結果
│
├── HIGH が 2領域以上 ?
│   └── Yes → [緊急] 産業医による総合健康レビュー
│       ├── 担当: 産業医
│       ├── 期限: 1週間以内
│       └── CL: 全領域レビュー → 就業適性再検討 → 専門医優先順位
│           → 事業者意見書 → 1ヶ月後再評価
│
├── 睡眠(non-LOW) かつ 疲労(non-LOW)
│   かつ 両方MODERATEではない ?
│   └── Yes → [緊急] 居眠り運転リスクの総合評価
│       ├── 担当: 産業医
│       ├── 期限: 1週間以内
│       └── CL: ヒヤリハット聞取り → 運転中眠気確認 → 休憩パターン見直し
│           → SAS検査照合
│
├── HIGH = 0 かつ MODERATE ≥ 2 ?
│   └── Yes → [推奨] 保健師による定期フォローアップ
│       ├── 担当: 保健師
│       ├── 期限: 1ヶ月以内
│       └── CL: 月1回FU面談 → 改善目標設定 → 3ヶ月後再評価
│
└── HIGH = 0 かつ MODERATE = 0 ?
    └── Yes → [任意] 次回定期健診・問診の実施
        ├── 担当: 保健師
        └── 期限: 次回定期健診時
```

**居眠り運転リスク判定の条件詳細**:

```
睡眠.riskLevel ≠ LOW
  AND 疲労.riskLevel ≠ LOW
  AND NOT (睡眠=MODERATE AND 疲労=MODERATE)
```

つまり、以下の組合せでトリガー:

| 睡眠 | 疲労 | 発火? |
|------|------|-------|
| HIGH | HIGH | Yes |
| HIGH | MODERATE | Yes |
| MODERATE | HIGH | Yes |
| MODERATE | MODERATE | No（単独対応で十分） |
| LOW | any | No |
| any | LOW | No |

## 6. 受診勧奨マトリクス

| 領域 | LOW | MODERATE | HIGH |
|------|-----|----------|------|
| **睡眠** | - | 睡眠外来/呼吸器内科（SASスクリーニング） | 睡眠専門医（PSG精密検査・CPAP） |
| **ストレス** | - | 産業医フォロー（任意） | 心療内科/精神科 |
| **疲労** | - | 産業医フォロー（任意） | 産業医面談（過重労働）+ 背景疾患精査 |
| **食事** | - | 特定保健指導候補 | 管理栄養士個別指導 / 特定保健指導 |
| **運動** | - | 保健師運動指導 | 運動指導 / 保健指導 / 特定保健指導 |

`referralNeeded = true` となるのは: **睡眠MODERATE以上、ストレスHIGH、疲労HIGH、食事HIGH、運動HIGH**
