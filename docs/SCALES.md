# DWA 使用尺度・学術的根拠

## 概要

DWA では5つの健康領域それぞれに、学術的に妥当性が検証された尺度を使用しています。
本ドキュメントでは、各尺度の学術的背景、妥当性検証の根拠、ドライバー職種への適用根拠を記載します。

---

## 1. 睡眠: ESS（エプワース眠気尺度）

### 1.1 尺度概要

| 項目 | 内容 |
|------|------|
| 正式名称 | Epworth Sleepiness Scale (ESS) |
| 日本語版 | JESS (Japanese ESS) |
| 開発者 | Johns MW (1991) |
| 項目数 | 8項目 |
| 回答尺度 | 4件法（0〜3点） |
| スコア範囲 | 0〜24点 |
| 評価対象 | 日中の眠気（主観的） |

### 1.2 質問項目

ESSは標準化された質問文を使用しており、日本語版の文言もそのまま採用しています。

| # | 質問文 | 回答選択肢 |
|---|--------|-----------|
| 1 | 座って読書をしているとき | 0:眠くならない 1:少し眠くなる 2:しばしば眠くなる 3:よく眠くなる |
| 2 | テレビを見ているとき | 同上 |
| 3 | 会議、映画館、劇場などで静かに座っているとき | 同上 |
| 4 | 乗客として1時間続けて自動車に乗っているとき | 同上 |
| 5 | 午後に横になって休息をとっているとき | 同上 |
| 6 | 座って人と話をしているとき | 同上 |
| 7 | 昼食後（飲酒なし）に静かに座っているとき | 同上 |
| 8 | 自分で車を運転中に、信号や交通渋滞で数分間止まっているとき | 同上 |

### 1.3 カットオフ値の根拠

| スコア | 判定 | 根拠 |
|--------|------|------|
| 0〜10点 | 正常 | Johns (1991) の原著で正常範囲として設定 |
| 11〜15点 | 過度の眠気 | SASスクリーニングの感度が高い範囲（Asaoka et al., 2010） |
| 16点以上 | 重度の眠気 | 睡眠障害の診断的意義が高い、専門医紹介の明確な基準 |

### 1.4 学術文献

1. **Johns MW.** A new method for measuring daytime sleepiness: the Epworth sleepiness scale. *Sleep*. 1991; 14(6): 540-545. doi: 10.1093/sleep/14.6.540
   - ESS原著論文。日中眠気の定量的評価法として開発。

2. **Takegami M, Suzukamo Y, Wakita T, et al.** Development of a Japanese version of the Epworth Sleepiness Scale (JESS) based on Item Response Theory. *Sleep Med*. 2009; 10(5): 556-565. doi: 10.1016/j.sleep.2008.04.015
   - 日本語版JESS。項目反応理論に基づく妥当性検証。

3. **Asaoka S, Noda A, Inoue Y.** Investigation of obstructive sleep apnea and excessive daytime sleepiness in professional Japanese truck drivers. *Sleep Biol Rhythms*. 2010; 8(1): 21-29.
   - 日本人大型トラック運転者（n=1,720）を対象にESSの有用性を検証。ドライバー集団でのカットオフ値の妥当性を確認。

### 1.5 ドライバー職種への適用根拠

- Asaoka et al. (2010) で日本人トラックドライバー1,720名を対象に妥当性を検証済み
- 質問8「自分で車を運転中に…止まっているとき」は運転者にとって直接的な関連性がある
- SASスクリーニングの第一段階として国土交通省の健康管理マニュアルでも推奨
- 本ツールでは質問の前文でドライバー業務を意識した回答を促す補足を追加（質問文自体は標準化文言を維持）

## 2. ストレス: K6（ケスラー心理的苦痛尺度）

### 2.1 尺度概要

| 項目 | 内容 |
|------|------|
| 正式名称 | Kessler Psychological Distress Scale (K6) |
| 開発者 | Kessler RC et al. (2002) |
| 項目数 | 6項目 |
| 回答尺度 | 5件法（0〜4点） |
| スコア範囲 | 0〜24点 |
| 評価対象 | 非特異的心理的苦痛（うつ・不安） |
| 回答期間 | 過去30日間 |

### 2.2 質問項目

| # | 質問文 | 回答選択肢 |
|---|--------|-----------|
| 1 | 神経過敏に感じましたか | 0:まったくない 1:少しだけ 2:ときどき 3:たいてい 4:いつも |
| 2 | 絶望的だと感じましたか | 同上 |
| 3 | そわそわ、落ち着かなく感じましたか | 同上 |
| 4 | 気分が沈み込んで、何が起こっても気が晴れないように感じましたか | 同上 |
| 5 | 何をするのも骨折りだと感じましたか | 同上 |
| 6 | 自分は価値のない人間だと感じましたか | 同上 |

### 2.3 カットオフ値の根拠

| スコア | 判定 | 根拠 |
|--------|------|------|
| 0〜4点 | 正常 | Furukawa et al. (2008)、Sakurai et al. (2011) |
| 5〜12点 | 心理的苦痛あり | 気分・不安障害のスクリーニング閾値 |
| 13点以上 | 重度の苦痛 | 重度精神障害（SMI）のスクリーニング基準。AUC 0.94（日本語版） |

### 2.4 学術文献

1. **Kessler RC, Andrews G, Colpe LJ, et al.** Short screening scales to monitor population prevalences and trends in non-specific psychological distress. *Psychol Med*. 2002; 32(6): 959-976. doi: 10.1017/S0033291702006074
   - K6原著論文。12カ国での開発・妥当性検証。

2. **Furukawa TA, Kawakami N, Saitoh M, et al.** The performance of the Japanese version of the K6 and K10 in the World Mental Health Survey Japan. *Int J Methods Psychiatr Res*. 2008; 17(3): 152-158. doi: 10.1002/mpr.257
   - WMH-J調査（n=2,436）でK6日本語版の妥当性を検証。AUC 0.94。

3. **Sakurai K, Nishi A, Kondo K, et al.** Screening performance of K6/K10 and other screening instruments for mood and anxiety disorders in Japan. *Psychiatry Clin Neurosci*. 2011; 65(5): 434-441. doi: 10.1111/j.1440-1819.2011.02236.x
   - 日本人集団でのK6カットオフ値の最適化研究。

### 2.5 ドライバー職種への適用根拠

- K6は厚生労働省「ストレスチェック制度」の推奨尺度の一つ
- 6項目と短く、面談の限られた時間でも実施可能
- 非特異的な心理的苦痛を測定するため、うつ・不安の両方をスクリーニング可能
- 日本語版は大規模疫学調査（WMH-J）で妥当性検証済み

## 3. 疲労: 疲労蓄積度自己診断チェックリスト（厚生労働省）

### 3.1 尺度概要

| 項目 | 内容 |
|------|------|
| 正式名称 | 労働者の疲労蓄積度自己診断チェックリスト |
| 開発元 | 厚生労働省 労働基準局 |
| 項目数 | 13項目（本ツールで使用する自覚症状に関する部分） |
| 回答尺度 | 4件法（0〜3点） |
| スコア範囲 | 0〜39点 |
| 評価対象 | 疲労の蓄積度 |
| 回答期間 | 最近1ヶ月 |

### 3.2 質問項目

| # | 質問文 | 回答選択肢 |
|---|--------|-----------|
| 1 | イライラする | 0:ほとんどない 1:ときどきある 2:よくある 3:いつもある |
| 2 | 不安だ | 同上 |
| 3 | 落ち着かない | 同上 |
| 4 | ゆううつだ | 同上 |
| 5 | よく眠れない | 同上 |
| 6 | 体の調子が悪い | 同上 |
| 7 | 物事に集中できない | 同上 |
| 8 | することに間違いが多い | 同上 |
| 9 | 仕事中、強い眠気に襲われる | 同上 |
| 10 | やる気が出ない | 同上 |
| 11 | へとへとだ（身体的に） | 同上 |
| 12 | 朝、起きた時にぐったりした疲れを感じる | 同上 |
| 13 | 以前と比べて、疲れやすい | 同上 |

### 3.3 カットオフ値の根拠

| スコア | 判定 | 根拠 |
|--------|------|------|
| 0〜4点 | 低蓄積 | 厚生労働省ガイドラインに基づく |
| 5〜14点 | 中程度蓄積 | 注意力低下リスク域 |
| 15点以上 | 高蓄積 | 過重労働面談の推奨域 |

### 3.4 学術文献

1. **厚生労働省.** 労働者の疲労蓄積度自己診断チェックリスト. 厚生労働省 労働基準局. 2023.
   - 過重労働による健康障害防止のための総合対策に基づく自己診断ツール。2023年改正版。

2. **厚生労働省.** 過重労働による健康障害防止のための総合対策について. 厚生労働省 基発. 2020.
   - 長時間労働者への面接指導制度の根拠通達。疲労蓄積度チェックリストの活用を推奨。

3. **Useche SA, Ortiz VG, Cendales BE.** Stress-related psychosocial factors at work, fatigue, and risky driving behavior in bus rapid transport (BRT) drivers. *Accid Anal Prev*. 2017; 104: 106-114. doi: 10.1016/j.aap.2017.04.023
   - バスドライバーにおける疲労蓄積と危険運転行動の関連を検証。

### 3.5 ドライバー職種への適用根拠

- 厚生労働省が過重労働対策として公式に推奨する尺度
- 長時間労働者への面接指導（安衛法第66条の8）の事前スクリーニングとして使用可能
- 改善基準告示に基づく拘束時間・運転時間の遵守状況と照合することで、制度的な対応が可能
- 質問の前文で「乗務中・乗務後の状態を思い浮かべてください」とドライバー文脈を補足

## 4. 食事・栄養: 特定健診質問票準拠

| 項目 | 内容 |
|------|------|
| 正式名称 | 食事評価（特定健診質問票・MLIT健康管理マニュアル準拠） |
| 構成根拠 | 厚生労働省「標準的な健診・保健指導プログラム」+ 国土交通省「運転者の健康管理マニュアル」 |
| 項目数 | 8項目 |
| 回答尺度 | 5件法（0〜4点、一部逆転項目） |
| スコア範囲 | 0〜32点 |
| 評価対象 | 食生活の質 |
| 特記事項 | **高得点 = 良好**（逆転スコアリング） |

### 4.2 質問項目（ドライバー最適化済み）

| # | 質問文 | ドライバー最適化ポイント |
|---|--------|----------------------|
| 1 | 朝食を毎日食べていますか（早朝出発の日も含めて） | 「早朝出発の日も含めて」を追加 |

| 3 | 1日の水分摂取量はどのくらいですか（運転中の摂取も含む） | 「運転中の摂取も含む」を追加 |
| 4 | 夕食後に間食をとることがありますか | 逆転項目 |
| 5 | 食事の時間は規則的ですか（シフト勤務・長距離運転の影響も含めて） | 「シフト勤務・長距離運転の影響」を追加 |
| 6 | 塩辛い食品を好んで食べますか | 逆転項目 |
| 7 | お酒を飲む頻度と量について | 逆転項目 |
| 8 | カフェイン飲料の1日の摂取量は（眠気覚まし目的を含む） | 「眠気覚まし目的を含む」を追加 |

### 4.3 学術文献

1. **厚生労働省.** 標準的な健診・保健指導プログラム【令和6年度版】. 厚生労働省 健康局. 2024.
   - 特定健診における食生活に関する質問票項目を参考に構成。

2. **国土交通省.** 事業用自動車の運転者の健康管理マニュアル. 国土交通省 自動車局. 2022.
   - ドライバーの食生活・飲酒・カフェイン摂取に関する指導事項を参考に構成。

### 4.4 ドライバー職種への適用根拠

- 特定健診の質問票をベースに、ドライバー特有の問題（早朝出発、運転中の水分、シフト勤務による食事時間の乱れ、眠気覚ましのカフェイン過剰摂取）を質問文に反映
- 国土交通省の健康管理マニュアルの指導事項と整合
- 質問文の変更は補足情報の追加にとどめ、測定構成概念（食生活の質）は変更していない

## 5. 運動・身体活動: IPAQ-SF

### 5.1 尺度概要

| 項目 | 内容 |
|------|------|
| 正式名称 | International Physical Activity Questionnaire - Short Form (IPAQ-SF) |
| 開発元 | IPAQ Research Committee |
| 項目数 | 7項目（3活動×2 + 座位時間） |
| スコア算出 | MET-分/週 |
| スコア範囲 | 0〜∞（上限なし） |
| 評価対象 | 身体活動量 |
| 回答期間 | 過去7日間 |

### 5.2 質問構造

| # | 質問文 | 選択形式 | ドライバー最適化 |
|---|--------|---------|----------------|
| 1 | 激しい身体活動の日数 | 0〜7日 | 「重い荷物の積み降ろし」を例示 |
| 2 | 激しい身体活動の1日平均時間 | 0〜120分以上 | - |
| 3 | 中程度の身体活動の日数 | 0〜7日 | 「車両点検作業」を例示 |
| 4 | 中程度の身体活動の1日平均時間 | 0〜120分以上 | - |
| 5 | 歩行の日数 | 0〜7日 | 「休憩中の散歩、業務中の歩行」を例示 |
| 6 | 歩行の1日平均時間 | 0〜120分以上 | - |
| 7 | 座位時間（平日） | 2〜13時間 | 「運転中の座位時間を含む」を追加 |

### 5.3 MET値と分類基準

**MET値（IPAQ公式ガイドライン準拠）**:

| 活動種別 | MET値 |
|---------|-------|
| 激しい身体活動 | 8.0 MET |
| 中程度の身体活動 | 4.0 MET |
| 歩行 | 3.3 MET |

**分類基準**:

| 分類 | MET-分/週 | IPAQ公式カテゴリ |
|------|----------|----------------|
| 低活動 | < 600 | Inactive |
| 中活動 | 600〜3,000 | Minimally Active |
| 高活動 | > 3,000 | HEPA Active |

### 5.4 学術文献

1. **Craig CL, Marshall AL, Sjöström M, et al.** International physical activity questionnaire: 12-country reliability and validity. *Med Sci Sports Exerc*. 2003; 35(8): 1381-1395. doi: 10.1249/01.MSS.0000078924.61453.FB
   - 12カ国での信頼性・妥当性検証。テスト-リテスト信頼性 rho=0.80。

2. **IPAQ Research Committee.** Guidelines for Data Processing and Analysis of the International Physical Activity Questionnaire (IPAQ) – Short and Long Forms. 2005.
   - IPAQ-SFの公式スコアリングガイドライン。MET値と分類基準の定義。

3. **Varela-Mato V, Cancela JM, Ayan C, et al.** Lifestyle and health among Spanish transportation drivers: a systematic review. *Int J Environ Res Public Health*. 2017; 14(5): 547. doi: 10.3390/ijerph14050547
   - 運輸業ドライバーの座位行動と健康リスクに関するシステマティックレビュー。

### 5.5 ドライバー職種への適用根拠

- IPAQ-SFは12カ国で妥当性が検証された国際標準尺度
- 本ツールでは数値直接入力ではなく、選択式に変換（ユーザビリティ向上）
  - 日数: 0〜7のボタン選択
  - 時間: 「していない」〜「2時間以上」の範囲選択（代表値でMET計算）
  - 座位: 2時間刻みの範囲選択（中央値でMET計算）
- ドライバーの業務活動（荷物積み降ろし、車両点検）を質問文の例示に追加
- 座位時間に「運転中の座位時間を含む」を明記（ドライバーは職業的に長時間座位）
- 座位時間≥8時間で追加警告を発出（エコノミークラス症候群リスク）

## 6. 尺度の妥当性に関する注意事項

### 6.1 質問文変更に関するポリシー

- **ESS（睡眠）**: 標準化された質問文をそのまま使用。前文に補足のみ追加。
- **K6（ストレス）**: 標準化された質問文をそのまま使用。変更なし。
- **疲労蓄積度CL**: 標準化された質問文をそのまま使用。前文に「乗務中・乗務後」の補足を追加。
- **食事評価**: 特定健診質問票をベースに、ドライバー文脈の補足情報を括弧内に追加。測定構成概念は不変。
- **IPAQ-SF**: 質問の構造は維持。数値入力を選択式に変換（回答精度は選択肢の代表値に依存）。

### 6.2 限界

1. **自己記入式**: すべての尺度が自己報告に基づくため、過少・過大報告のバイアスが存在
2. **横断評価**: 1時点のスクリーニングであり、経時変化の評価には再実施が必要
3. **食事評価**: BDHQ等の詳細な食事調査と比べて項目数が限定的
4. **IPAQ-SF選択式変換**: 時間を範囲選択にした影響でMET計算の精度が原著のフリーテキスト入力より低下する可能性
5. **集団特性**: カットオフ値は一般集団で設定されたものであり、ドライバー集団固有の最適値は今後の検証が必要
